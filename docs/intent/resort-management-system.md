# Intent: Resort Management System

- **Outcome:** A complete booking management system for a Thai resort covering room reservations, boat tours, package deals, and payment processing
- **User:** Resort owner/staff (admin, room staff, boat staff) and customers (guests, members)
- **Why now:** The resort needs a modern digital system to replace manual processes
- **Success:** All 5 roles can perform their core functions (browse, book, pay, manage, report) without resorting to paper or external tools
- **Constraint:** Solo developer, real production system for a paying client, must be deployable on Vercel serverless
- **Out of scope:** Mobile apps, real-time WebSocket, multi-language support, multi-resort SaaS, offline mode
